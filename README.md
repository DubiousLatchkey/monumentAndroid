# Monumental
## Tagging monuments with firebase

Monumental uses uses Firebase to connect people to a database of monuments around the world.  When they get near one, they can take a picture of it.  If the picture is recognized by the AI as the monument in question, it checks in the user to the monument and rewards them.  You can also pay balloon hunt, where each person can place a balloon with a photo that people nearby can see and find for rewards.
